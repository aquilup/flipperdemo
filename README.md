# Flipper Zero — Premium Product Experience

Cinematic dark landing page for Flipper Zero: Next.js 15, React Three Fiber, GSAP ScrollTrigger, Lenis, Framer Motion.

## Stack

- Next.js 15 + TypeScript + Tailwind CSS
- Three.js / R3F / Drei
- GSAP + ScrollTrigger + `@gsap/react`
- Lenis smooth scroll
- Framer Motion
- Lucide + react-icons

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Notes

- Procedural 3D Flipper model (no external GLTF required)
- `prefers-reduced-motion` disables heavy motion / Lenis
- SEO: Open Graph, Twitter cards, Schema.org Product JSON-LD
- Educational / research framing only
